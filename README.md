# 物業停車智慧儀表板

## 專案介紹

這是一個物業停車場智慧管理系統的MVP版本，提供實時監控、數據可視化和風險預警功能。

## 功能特點

### 📊 物業看板
- 實時車位狀態監控
- 預估等待時間計算
- 車位狀態占比圖表
- 等待時間趨勢分析
- 違規停車提醒
- 車輛剮蹭告警
- 異常提醒總覽

### 📝 門崗上報
- 可用車位數量上報
- 排隊車輛數量記錄
- 平均放行速率統計
- 上報人員信息
- 備註信息記錄

### 📈 歷史趨勢
- 歷史數據查詢
- 趨勢分析
- 狀態變化追蹤

## 技術棧

- **前端**: 純HTML5 + CSS3 + JavaScript
- **圖表庫**: ECharts 5
- **數據存儲**: LocalStorage
- **響應式設計**: 支持桌面和移動端

## 使用說明

1. 直接在瀏覽器中打開 `index.html` 即可使用完整功能
2. 各個子頁面可單獨使用，適合導入到墨刀等設計工具
3. 數據會自動保存在瀏覽器的LocalStorage中

## KPI指標

- **空閒狀態**: 可用車位 > 20% 總車位
- **緊張狀態**: 5% ≤ 可用車位 ≤ 20% 總車位
- **無車位狀態**: 可用車位 < 5% 總車位
- **等待時間預警**: 預估等待時間 ≥ 15 分鐘

## 風險預警

- **違停風險**: 排隊車輛 ≥ 8 輛
- **剮蹭風險**: 無車位且排隊車輛 ≥ 10 輛
- **高風險**: 排隊車輛 ≥ 12 輛
